# PluginInfo

Info for a plugin.

| FIELD | TYPE | DESCRIPTION |
| ----- | ---- | ----------- |
| name | string | The Plugin Name |
| description | string | The Plugin Description |
| version? | `${number}.${number}.${number}` (ex: 0.0.1, 1.0.0, 3.1.4) | The Plugin Version |
| author? | string | The Plugin Author |
